# Optiland Optional Manual Validation Reports

This directory stores optional manual Optiland validation reports.

Default tests, smoke checks, quality gates, CI, and release validation do not
execute Optiland. Do not commit large output files from `/tmp`, do not make
Optiland a default dependency, and do not use these reports to claim
production-grade optical validation.

The 2026-05-20 optional micro-benchmark evidence is recorded in
`optiland_micro_benchmark_2026-05-20.md` as optional manual ray/path smoke
evidence only. It does not claim optical correctness, production-grade physical
validation, or a formal convergence proof.
