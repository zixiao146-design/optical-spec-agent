# Optical Calculators Examples

These local JSON fixtures exercise preview/design-assist calculators only.

- No external solver execution is performed.
- No external LLM is called.
- No network or material database lookup is performed.
- Results are not production-grade physical validation.
- Formal convergence proof is not claimed.

## Fixtures

- `thin_film_ar_coating.json`: simple anti-reflection coating preview.
- `paraxial_lens_imaging.json`: thin-lens imaging preview.
- `gaussian_beam_propagation.json`: Gaussian beam propagation preview.
- `slab_waveguide_v_number.json`: slab waveguide V-number preview.
