# Photonic Crystal Band Preview

Design goal: preview a photonic crystal band workflow with MPB-oriented scaffold output.

- Suggested materials: Si, GaAs, SiO2, air.
- Suggested adapter: MPB.
- Workflow steps: parse, validate, material suggestion, workflow plan, adapter preview, human review.
- Expected next action: verify lattice constants and material constants before any physical interpretation.

Safety boundaries:
- No solver is executed by default.
- No external LLM is called by default.
- No production-grade physical validation is claimed.
- Formal convergence proof is not claimed.
