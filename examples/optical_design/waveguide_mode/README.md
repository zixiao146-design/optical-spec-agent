# Waveguide Mode Preview

Design goal: preview waveguide mode setup for local design review.

- Suggested materials: Si, SiO2, Si3N4.
- Suggested adapter: MPB for band/eigenmode-style preview; Elmer remains Level 2 + Level-3-ready, install deferred.
- Workflow steps: parse, validate, material suggestion, workflow plan, adapter preview, human review.
- Expected next action: confirm geometry and boundary assumptions before any optional solver work.

Safety boundaries:
- No solver is executed by default.
- No external LLM is called by default.
- No production-grade physical validation is claimed.
- Formal convergence proof is not claimed.
