# Lens Raytrace Preview

Design goal: preview a local lens/ray optics workflow with Optiland-oriented scaffold output.

- Suggested materials: BK7 preview, fused silica preview, air.
- Suggested adapter: Optiland.
- Workflow steps: parse, validate, material suggestion, workflow plan, adapter preview, human review.
- Expected next action: verify glass catalog data and prescription fields before physical conclusions.

Safety boundaries:
- No solver is executed by default.
- No external LLM is called by default.
- No production-grade physical validation is claimed.
- Formal convergence proof is not claimed.
