# Agent Studio Quickstart Examples

These examples support the local Agent Studio quickstart.

- `nanoparticle_demo_spec.json` points to the offline minimal nanoparticle spec
  used by the local validate and adapter-preview demos.
- `quickstart_workflow_request.json` points to the offline local workflow
  preview request.

Safety boundaries:

- No network access is required.
- No solver is executed.
- No external LLM is called.
- No PyPI/TestPyPI upload is performed.
- No tag or GitHub release is created.
- Preview artifacts are not production-grade physical validation.
- Formal convergence proof is not claimed.
