# v1.0 PyPI Decision Gate

## Current state

- TestPyPI verified: yes, for 0.9.0rc6.dev0
- PyPI published: no
- PyPI approval: not granted

## Decision options

- GitHub-only v1.0.
- PyPI after v1.0 GitHub release.
- PyPI together with v1.0 release.
- PyPI delayed until post-v1.0.

## Recommendation

- Keep PyPI publication separate and explicitly approved.
- Do not publish PyPI automatically as part of planning.
- Use the PyPI publication readiness checklist before any upload.

## Required approval phrase

PyPI publication requires an explicit maintainer approval such as:

> I approve PyPI publication for optical-spec-agent version X.
